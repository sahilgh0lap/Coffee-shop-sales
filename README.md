# Coffee Shop Sales Dashboard 

Interactive Power BI dashboard built to analyze coffee shop sales, customer footfall, product performance, and ordering trends.  
Features KPI cards, slicers, and interactive visualizations for sales by hour, product category, drink size, weekdays, store locations, and top-selling products.  
Developed using **Power BI**, **Power Query**, **DAX**, and **Data Modeling** to provide actionable business insights and support data-driven decision-making.

**Tech Stack:** Power BI • Power Query • DAX • Data Modeling
